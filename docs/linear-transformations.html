1 Linear Transformations in \(\mathbb{R}^2\)

A linear transformation \(T:\mathbb{R}^2\to\mathbb{R}^2\) can be represented by a matrix

\[ A=\begin{pmatrix} a & b \\ c & d \end{pmatrix}. \]

Using column vectors, the transformation is

\[ T\begin{pmatrix}x\\y\end{pmatrix} = A\begin{pmatrix}x\\y\end{pmatrix} = \begin{pmatrix} ax+by \\ cx+dy \end{pmatrix}. \]

The first column of \(A\) is where the basis vector \(e_1=\begin{pmatrix}1\\0\end{pmatrix}\) goes. The second column is where \(e_2=\begin{pmatrix}0\\1\end{pmatrix}\) goes:

\[ A e_1=\begin{pmatrix}a\\c\end{pmatrix}, \qquad A e_2=\begin{pmatrix}b\\d\end{pmatrix}. \]

The determinant is a compact summary of what the map does to area and orientation:

\[ \det A = ad-bc. \]

If \(\det A>0\), orientation is preserved. If \(\det A<0\), orientation is reversed. If \(\det A=0\), the plane collapses onto a line or point, so the transformation is not invertible.
